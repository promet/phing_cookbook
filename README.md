Phing Cookbook
==============
***
Cookbook dependency for [Drupal Audit](https://github.com/promet/drupal_audit).

##To Use
***
###**Recommended:** 
Simply reference this cookbook inside Berksfile:

`cookbook "phing_cookbook", git: "https://github.com/promet/phing_cookbook.git"`

###**Alternatively:** 
Clone repo. 

Run "vagrant up".
